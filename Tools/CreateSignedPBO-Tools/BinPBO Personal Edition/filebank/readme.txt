FileBank
==========================
Copyright (c) 2011 Bohemia Interactive a.s. All rights reserved.

   FileBank {options} source [source]

   source may be directory name or log file name (.log)

   Options:
   -encrypt encryption_name password_file
   -property name=value
   -property prefix=path to be set for files in pbo file
   -exclude filename    File with list of masks of files and directories to leave a packing into PBO.
